# Stock Market Dashboard (React + Express + Socket.io)

## Project Overview
This project is a Real-Time Stock Market Dashboard built using ReactJS, ExpressJS, and Socket.io.  
It demonstrates component styling, lifecycle methods, controlled/uncontrolled components, and real-time WebSocket updates.

---

## Features Implemented

### Styling & Bootstrap
- Bootstrap UI components (cards, buttons, layout)
- Custom CSS styling
- Light/Dark theme toggle

### Component Lifecycle Methods
- `componentDidMount()` for fetching initial stock data
- `componentDidUpdate()` for updating stock data when symbol changes

### Controlled & Uncontrolled Components
- Controlled Input: Stock Symbol Search
- Uncontrolled Input: Save Previous Search History (using `ref`)

### Real-Time Stock Price Updates (WebSockets)
- Express server streams real-time stock prices
- Socket.io sends updates every 2 seconds to React frontend


- Stock Trend Chart using Chart.js

---

- Frontend: ReactJS, Bootstrap, Chart.js
- Backend: Node.js, ExpressJS
- Real-Time: Socket.io
